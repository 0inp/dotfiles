# Triage Labels

Default GitHub labels for issue triage:

| Label | Purpose |
|-------|---------|
| `bug` | Confirmed bugs or reports of broken functionality. |
| `enhancement` | Feature requests or improvements. |
| `documentation` | Documentation updates or fixes. |
| `good first issue` | Beginner-friendly tasks. |
| `help wanted` | Tasks requiring community contributions. |
| `question` | Questions or discussions. |
| `wontfix` | Issues that will not be addressed. |
